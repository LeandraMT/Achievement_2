Documentation for leMoovie Application

Purpose

The application will be called 'leMoovie' and it will provide users with access to information about all types of movies, directors, actors and genres.

Some of the main features of the application will be for the users to receive information on movies, directors, actors and genres.
This allows the users to stay up to date to new launches or learn more about the movies.
The users will also be able to create and design their profile so they can save their data about their favourite movies.

  1. Home page with all the movies available
  2. Obtain data such as description of the movie, genre, image URL etc.
  3. Allow new users to register
  4. Allow users to update their info such as; username, password, email, date of birth etc.
  5. Allow existing users to deregister
  6. Allow users to add and remove movies
